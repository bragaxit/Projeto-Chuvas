# Projeto-Chuvas
Este projeto foi desenvolvido para ser uma ferramenta de prevenção e segurança. A ideia é transformar dados climáticos (temperatura e volume de chuva) em informações úteis. Ele serve para alertar o usuário sobre riscos de infraestrutura (como alagamentos e falta de luz) e sugerindo cuidados pessoais, garantindo que não seja pego de surpresa.
